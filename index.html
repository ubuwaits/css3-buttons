CSS3 buttons

Chad Mazzola created this collection of buttons that show what is possible using CSS3 while also maintaining the simplest possible markup. I updated his button.css file to add support for Opera linear gradients. They look best in WebKit-based browsers, almost as good in Firefox, with all other browsers falling back to a less-styled button. If you use these buttons in the wild, drop Chad a note and let him know.

Chad’s latest code is located at: http://github.com/ubuwaits/css3-buttons.

Classic buttons

This is the original set of buttons, showing a variety of CSS3 styles in different combinations. Study the code, then adjust to fit the context of where it's used.
