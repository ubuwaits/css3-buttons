CSS3 buttons

Designed & built by Chad Mazzola with inspiration from across the web.

This is a collection of CSS3 buttons that use the simplest possible markup. They look best in WebKit-based browsers, almost as good in Firefox, with all other browsers falling back to a less-styled button. If you use these buttons in the wild, drop me a note and let me know.

These buttons are now implemented using Sass, using Bourbon. The plain, generated CSS for all the buttons is also still available in its previous location within the repo.

The latest code is located at: http://github.com/ubuwaits/css3-buttons
